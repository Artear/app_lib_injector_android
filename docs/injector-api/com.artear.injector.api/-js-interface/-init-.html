JsInterface(key: String)

Use them to establish a suitable communication between a single javascript event for your web loaded in a webview and an object in your application, making a clean code and a single responsibility.

The ArtearProcessor use this annotation to generate an extra file for execute a traditional JavascriptInterface used in webview.

Note that the process register each class annotated whit it.

Parameters

key - the javascript event name
