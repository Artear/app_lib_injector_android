com.artear.injector.api.JsEventManager

A JsEventManager is an entity that control the communication between the webview and your javascript events. It is part of divide each responsibility idea.
com.artear.injector.api.JsInterface

Use them to establish a suitable communication between a single javascript event for your web loaded in a webview and an object in your application, making a clean code and a single responsibility.
