JsEventManagerProcess

class JsEventManagerProcess : Process<JsEventManagerClass>

Use JsEventManager and process them.

The aim is extend a simple function for initialize them and add each JsInterface like a new command. Also use a function addJavascriptInterfaces to attach the js flow events between the webview and that manager.
